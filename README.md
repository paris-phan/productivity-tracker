# Device Usage Logger

**Goal:** Track device usage across apps, websites, and idle time for comprehensive time analysis.

---

## Overview
A cross-platform logger for macOS, iOS, and Windows to track app usage, website activity, and user interaction. Data syncs to a central server for processing and insights to help with user productivity

### Key Features
- Track time on apps and websites.
- Detect idle periods.
- Works on macOS, iOS, and Windows.
- Syncs data to the cloud.

---

## Project Structure
```
/ios          # iOS logger
/macos        # macOS logger
/windows      # Windows logger

```
